声明块是由一个或多个声明组成，每隔声明则是一个属性-值对

```css
选择器{声明块}
举个例子
h1{color:red}
color属性

```

类选择器和ID选择器

### 类选择器

如果是应用样式而不去考虑涉及元素，就可以使用类选择器。

```
.className{}
```

### ID选择器

ID选择器只会使用一次，ID选择器不能像类选择器一样结合使用，因为ID选择器不可以有空格的词列表。

```
#IdName{}
```

类选择器和ID选择器可以同名
